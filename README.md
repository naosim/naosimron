# naosimron
自分のブログ用リポジトリです。
forkされても困ります。

## 構想
- jekyll等のジェネレータを使ってスタティックなサイトでつくる
- マスターにコミットしたら公開されるようにする(どこからでもかける)
- 文章はmarkdownで書く
- デザイン変更に強くする
- レスポンシブにする

## ブログとしての機能
- タグ
- カテゴリは無し
- RSS
## Hexoでの投稿メモ
### Hexo Serverを起動する
```
cd myblog
../node_modules/hexo/bin/hexo server
```
